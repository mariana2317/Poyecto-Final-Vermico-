# Poyecto-Final-Vermico-
se trata de un sistema de monitorio de humedad para vermicompost usando tecnología Arduino, este proyecto registra valores del sensor (calibrado con equipo Hana), procesa los datos en phyton, clasifica los niveles de humedad (SECO, MEDIO, OPTIMO, EXCESO), y genera visualizaciones para evaluar el estado de vermicompost. El sistema incluye lectura desde Arduino,indicadores luminosos (LED rojo,naranaja,verde y luz intermitente cuando se encuentra en exceso) segun el nivel de humedad , y un programa en phyton que limpia datos , permite ine¿gresar nuevas lecturas y genera resultados automaticos.
